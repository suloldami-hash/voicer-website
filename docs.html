Getting Started

Hardware-Grade Setup

Once you've downloaded the universal binary, drag Voicer to your Applications folder. On first launch, macOS Sequoia will request Accessibility permissions. This is required for the Bulletproof Auto-Paste engine to detect the active window.

API Configuration

Voicer requires a Whisper-compatible API key. Open the glass settings panel (Shift + Command + ,) and input your key. We support OpenAI and Groq for sub-second transcription speeds.

Precision Auto-Paste

Our focus detection system monitors the macOS window server to ensure text is only injected when a cursor is active. If you release the dictation hotkey while focused on a non-text element, Voicer will automatically copy the text to your clipboard instead.

  • Sub-10ms focus detection
  • Intelligent fallback to clipboard
  • Support for all native macOS applications
